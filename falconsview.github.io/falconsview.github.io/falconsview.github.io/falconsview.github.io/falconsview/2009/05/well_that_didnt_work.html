Shawn:

From looking at the source, it's not really an MT problem, per se. It's using div classes and span classes for formatting. Works well in a browser since the browser is also pulling the style sheet but not in a reader where the style sheet is absent. This is a problem with Wordpress, too. The only solution I've found is to use the HTML editor rather than the Rich Text editor for posts. That way you can format with HTML tags that don't need a style sheet.
